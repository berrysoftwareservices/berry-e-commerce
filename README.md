# Berry Project
Welcome to Berry Project

Instructions:

1. Clone the repo
   ```sh
   git clone https://github.com/berrysoftwareservices/berry-e-commerce.git
   ```
2. Go to project folder
    ```sh
    cd berry-e-commerce
     ```
3. Install yarn packages
   ```sh
   yarn
   ```
4. Run the app
   ```sh
   yarn dev
   ```
